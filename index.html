How to use?

Encoding

Write your message in the "Encoding" section.
The whole input should look like this example:

301:elephants are green

Copy the encoded message from the "output" box.

Decoding

Paste the message in the "Decoding" section.
The message you are trying to decode should look like this:

301:szs3wn165-n4s-u4ss1

If the key match the message, decoded message will appear in the "output" box.
